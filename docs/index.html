OpenQuantArena

Paper-trading only. Strategy Lab + Backtest Worker + Leaderboard.

FastAPI + Postgres + Redis + Celery. Deterministic dummy backtests (demo-ready).

GitHub Repo Quick Demo (README)

What it does

  • Users: email + password + JWT auth
  • Strategies: CRUD + upload a .py file (stored in a volume)
  • Backtests: enqueue a job (Celery) over a date range
  • Results: store metrics (CAGR, max drawdown, Sharpe)
  • Arena: leaderboard on latest successful backtest per strategy

API UI

When deployed, FastAPI provides Swagger UI at /docs and OpenAPI JSON at /openapi.json.
